## 如何使用

方法一：把[PPI 计算器](ppi.html)的代码下载下来用浏览器打开即可。

方法二：在自己电脑上新建一个html文件，复制[PPI 计算器](ppi.html)的代码粘贴进去保存即可，然后用浏览器打开即可。
